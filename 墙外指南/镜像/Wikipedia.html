Proxy Free Chinese Wikipedia 无需代理的中文维基百科 ​

通过 CloudFlare Workers 实现中国大陆网络环境直接访问

官网描述

海纳百川·有容乃大

我们通过先进的 CloudFlare Workers 边缘计算技术构建了一个稳定、高速、零延迟的中文维基百科访问网关，无需任何代理工具即可自由探索人类知识的海洋。

  • 零配置访问 — 无需安装 VPN、代理或任何插件，打开即用，纯净体验；
  • 全球边缘加速 — 依托 CloudFlare 300+ 全球节点，智能路由，毫秒响应，告别卡顿；
  • 内容实时同步 — 与维基百科官方源站保持 100% 内容一致，无缓存延迟，知识永新； 1
  • 完整功能支持 — 搜索、编辑预览、图片加载、多语言切换等功能一应俱全，体验与原站无异。

安全可靠

所有访问请求均通过 CloudFlare 加密通道传输，无用户数据留存，不记录访问日志，保障您的隐私与数据安全；支持 HTTPS 全站加密，杜绝中间人攻击。

全球加速

依托 CloudFlare 全球 300+ 边缘节点，智能路由就近访问，无论您身处北京、广州还是海外均能享受毫秒级响应速度，告别卡顿与超时。

高速稳定

采用边缘缓存策略，静态资源缓存 30 天、HTML 页面缓存 15 分钟，大幅降低源站压力、提升加载速度。即使高峰时段，依然稳定如初。

智能改写

自动识别并重写页面内所有资源链接（图片、脚本、样式表），确保维基百科完整功能可用；支持 srcset、data-src、内联样式等复杂场景，页面渲染零缺失。

零服务器运维

基于 Serverless 架构，无需购买、配置或维护任何服务器。自动弹性伸缩，按需计费，成本近乎为零，个人开发者也能轻松部署全球级服务。

公益开源

本项目由开发者杖雍皓个人维护，非商业用途，代码已经在 GitHub 开源，欢迎学习、复用、共建，共同推动知识自由访问。
